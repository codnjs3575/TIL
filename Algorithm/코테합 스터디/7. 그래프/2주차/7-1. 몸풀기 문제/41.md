- 노드 0 [[1, 4], [2, 3], [4, -6]]

  - 0 -> 1 : 4
    - 중간노드 1) 1 -> 3 : 5
  - 0 -> 2 : 3
    - 중간노드 2) 2 -> 1 : 2
  - 0 -> 4 : -6
    - 중간노드 4) 4 -> 2 : 2

- 노드 1 [[3, 5]]

  - 1 -> 3 : 5
    - 중간노드 3) 3 -> 0 : 7 | 3 -> 2 : 4

- 노드 2 [[1, 2]]

  - 2 -> 1 : 2
    - 중간노드 1) 1 -> 3 : 5

- 노드 3 [[0, 7], [2, 4]]

  - 3 -> 0 : 7
    - 중간노드 0) 0 -> 1 : 4 | 0 -> 2 : 3 | 0 -> 4 : -6
  - 3 -> 2 : 4
    - 중간노드 2) 2 -> 1 : 2

- 노드 4 [[2, 2]]

  - 4 -> 2 : 2
    - 중간노드 2) 2 -> 1 : 2
